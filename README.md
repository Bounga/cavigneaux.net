www.cavigneaux.net
==================

This is [Nicolas Cavigneaux][1]'s resume website, powered by [Nanoc][2].

[CoffeeScript][7], [SCSS][8], [HAML][9] and a [Guard file][10] are used to ease development and deployment.

The source is hosted on [BitBucket][5].
